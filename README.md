# STATLAB EXPERIMENTS

Plan and interpret A/B tests with statistical rigor and visual clarity.

---

## 📊 Overview
StatLab Experiments is a high-fidelity, stateless analytical tool designed for Product Engineers and Data Analysts. It transforms frequentist statistical theory into a clean, actionable, and visually premium interface, allowing for rapid decision-making in A/B testing cycles.

## 🚀 Key Capabilities
- **Experiment Planning**: Calculate sample size requirements based on baseline conversion, MDE, alpha, and power.
- **Statistical Analysis**: Run frequentist tests (Z-test for proportions) with automated interpretation.
- **Decision Support**: Three-state decision engine (Winner Probable, Inconclusive, Weak Effect).
- **Practical Significance**: Automated guardrails to distinguish between statistical significance and practical impact.
- **Reporting**: One-click professional report generation for Slack, Jira, or Notion.

## 🛠 Tech Stack
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui, Recharts.
- **Backend**: FastAPI, SciPy, statsmodels (Stateless architecture).
- **Design**: Minimalist, high-typography focus, optimized for "Apple-esque" restraint.

## 🚀 Quick Start
1. Ensure Python 3.11+ and Node.js are installed.
2. Run `start.bat` in the project root.
3. The script will automatically set up the virtual environment, install dependencies, and launch the stack.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
